About This Application

Greetings! Thank you for visiting the homepage of the KG To-Do App. This webpage features a to-do application designed to help individuals stay focused and organized. I originally developed this application for myself during college, and it helped me immensely. Now, I have upgraded it in hopes that it can help others achieve the same success.

The web application was developed using React.js for the frontend and Express.js for the backend. The desktop application was built using C++ along with libraries that enable communication with SQL databases. The database is managed by PostgreSQL on a small server computer I use for projects like this.
